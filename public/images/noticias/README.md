# Fotografias oficiais da INNEURO

Adicione nesta pasta somente fotografias reais e autorizadas da INNEURO, sem
pacientes, prontuários, laudos, telas ou informações identificáveis.

- Formatos preferenciais: WebP ou AVIF.
- Proporção recomendada: 16:9.
- Resolução recomendada: 1600 × 900.
- Tamanho máximo recomendado: 500 KB por arquivo otimizado.
- Informe título, descrição, categoria e texto alternativo aprovados junto de
  cada fotografia para cadastro em `src/data/company-highlights.ts`.

Não use bancos de imagens, imagens geradas, capturas de terceiros ou fotografias
de outras clínicas.
