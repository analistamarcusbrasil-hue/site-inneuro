# Assets fornecidos pela INNEURO

Integração realizada em 23/07/2026 a partir do pacote encaminhado pela INNEURO.

- A logo institucional é usada no cabeçalho, no menu móvel e no rodapé.
- A fotografia da fachada é o primeiro slide do carrossel da Home.
- As marcas de convênios e da parceria AmorSaúde são usadas como fallback estático enquanto a biblioteca de mídia do CMS não possui um arquivo correspondente.
- Os arquivos de marcas foram fornecidos pela INNEURO, mas não foram verificados independentemente nas centrais oficiais das operadoras.
- Os PNGs de marcas foram preservados sem recorte, recoloração, filtro ou remoção do fundo incorporado.

## Divergência de cadastro

`CAPE Saúde — CAPESESP` foi cadastrada como item próprio. O cadastro anterior `CAPSAÚDE` não foi substituído: permanece no arquivo de dados como pendente e inativo até confirmação específica da INNEURO.

## Preparação para o CMS

Os caminhos e metadados estão relacionados em `docs/assets-manifest.json`. Esses arquivos podem ser enviados futuramente à biblioteca de mídia do painel, sem que o site dependa desse upload para funcionar.
